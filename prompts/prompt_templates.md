# Prompt Templates — Mesoamerican Style

Use these as building blocks. Replace bracketed sections. Templates are organized by culture, output type, and compositional format.

---

## Universal Templates

### Template A — Bas-relief artifact (any culture)
`"[subject] as carved bas-relief stone, mesoamerican glyph borders, step-fret framing, mineral pigment in recesses (jade, cinnabar), raking light from low angle, chipped stone edges, codex-like composition, high-contrast incised outlines, [culture] style"`

### Template B — Codex page (Aztec/Mixtec)
`"codex page illustration of [scene], stacked horizontal registers, glyph blocks flanking the scene, flat mineral pigments (black, red, turquoise, yellow on cream), strong black outlines, no shading, symbolic proportions, border bands of step-fret or xicalcoliuhqui pattern, Aztec/Mixtec style"`

### Template C — Generative geometric pattern
`"procedural mesoamerican-inspired pattern, [step-fret/xicalcoliuhqui/interlocking scroll] bands, modular glyph tiles, [bilateral/radial] symmetry, [stone jade / mural red / codex] palette, crisp edges, carved channels, infinite tiling"`

### Template D — Shader-friendly abstract
`"abstract mesoamerican relief pattern, signed-distance-field aesthetic, beveled edges, groove shadows, tiled glyph blocks, low-frequency symmetry with high-frequency chisel noise, raking directional light, [jade green / cinnabar red / obsidian black] dominant"`
