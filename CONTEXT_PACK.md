# mesoamerican_style — Context Pack

Purpose: Provide structured, remixable context for "vibecoded" generative outputs (AI art prompts, shader ideas, and JavaScript generative design). This repo is intentionally *not* an app; it is a context/database repo meant to be mixed with other repos.

What to use this for:
- Prompt engineering for Mesoamerican-inspired imagery and symbols
- Visual constraints (motifs, composition rules)
- Color palettes grounded in stone, jade, obsidian, cinnabar, maize
- Generative systems mappings (glyph tiling, step-fret geometry, relief carving normals)

Recommended consumption order:
1. STYLE_GUIDE.md
2. motifs/motifs.json
3. palette/palettes.json
4. prompts/prompt_templates.md + prompts/negative_prompts.md
5. generative/shader_hints.md + generative/js_visual_patterns.md
